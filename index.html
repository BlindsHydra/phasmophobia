About

Phasmophobia SaveData is a project STRICTLY created to let users who play Phasmophobia use all the equipment and play what the real game is like without going through all the tricky steps. We allow our users to save a data file so that others can use different kinds of skills. These files are manually reviewed by a human and posted after inspection.
